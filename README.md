# üîß Run Locally

Clone the project

```bash
  git clone git@github.com:https://github.com/Joseane-Guedes/dashgo.git
```

Go to the project directory

```bash
  cd dashgo
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```
